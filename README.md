# 🎧 Podcast Player

Podcast Player 是一個基於 Vue.js 構建的單頁應用（SPA），提供 Podcast 播放功能，並支援進度控制與自動播放下一集的功能。

---

## 🚀 開啟專案方式

1. **Clone 專案**
   ```sh
   git clone https://github.com/iamCeliatw/podcast-player.git
   cd podcast-player
   ```
2. **安裝依賴**

   ```sh
   npm install
   ```

3. **啟動開發伺服器**

   ```sh
   npm run dev
   ```
